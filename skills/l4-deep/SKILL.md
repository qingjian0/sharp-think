# L4 深度分析

> 问题明确，根因清楚，方案多样，推荐具体，路径清晰，风险可控。

## 适用场景

满足以下条件：
- 5+ 步推理完成
- 需要多方案对比（至少 2 个）
- 决策影响中等偏高
- **推荐子任务拆分**

## 推理模板

```
## 问题定义
[精确描述要解决的问题，避免 XY Problem]

## 根因分析
[用 5-Why 思路，至少追问 3 层]

## 方案探索
- 方案 A：[名称]
  - 做法：[简要步骤]
  - 优势：[2-3 个]
  - 劣势：[1-2 个]
- 方案 B：[名称]
  - 做法：[简要步骤]
  - 优势：[2-3 个]
  - 劣势：[1-2 个]

## 推荐方案
[选择方案 X，理由：...]

## 实施路径
[分步骤执行计划，每步有可验证的产出]

## 风险预判
- 风险 1：[描述] → 应对：[措施]
- 风险 2：[描述] → 应对：[措施]
```

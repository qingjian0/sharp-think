# L5 全面研究

> 全面深入，系统思考，有据可依，风险可控。

## 适用场景

满足以下条件：
- 复杂决策，影响重大
- 需要多维度分析
- 涉及多个利益相关方
- **必须子任务拆分**
- **分阶段验证**

## 分阶段验证

L5 任务建议分阶段确认：
1. **理解确认** → 确认问题定义正确
2. **方案确认** → 确认方案方向正确
3. **实施确认** → 确认实施路径可行
4. **最终确认** → 确认完整输出满意

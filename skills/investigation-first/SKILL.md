# 🔎 调查研究

> "调查就像'十月怀胎'，解决问题就像'一朝分娩'。"

## 适用场景

- 要做决策但信息不足
- 对领域不熟悉，需要了解现状
- 需要收集多方意见

## 调查清单

- [ ] 现状是什么？（事实层面）
- [ ] 有哪些约束条件？
- [ ] 历史上是怎么做的？
- [ ] 别人是怎么做的？
- [ ] 用户/利益相关方有什么反馈？
- [ ] 有哪些已知的未知？
- [ ] 有哪些可依托的基础？

## 信息来源

根据具体情况选择：

- 文档和文件（项目文档、代码库
- 历史记录（历史对话、历史决策）
- 相关方沟通（用户、团队、利益相关者）
- 最佳实践（行业标准、同类项目）
- 实测结果（测试、实验、数据）

## 输出模板

```
## 调查结论
- 现状是：[事实描述]
- 关键约束是：[约束列表]
- 主要的未知项是：[未知列表]
- 已知的可依托基础是：[基础列表]

## 信息来源
- 来源 1：[描述]
- 来源 2：[描述]
```

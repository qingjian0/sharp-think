# L2 简要分析

> 结论先行，理由简要，不要过度展开。

## 适用场景

满足以下条件：
- 2-3 步推理可完成
- 涉及简单判断
- 有明确约束但不需要多方案

## 推理模板

```
## 核心结论
[一句话给出结论]

## 简要理由
[2-3 句话说明理由]
```

## 输出模板

使用 `templates/l2-output.md`。

---

**示例**：

用户：帮我看一下这个方案可行吗
AI：
## 核心结论
方案可行，但需要调整两个细节。

## 简要理由
1. 整体逻辑是通顺的
2. 但时间安排偏紧，建议延期 2 天
3. 预算分配需要调整一下

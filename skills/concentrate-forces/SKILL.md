# 🎯 集中兵力

> "不打无准备之仗，不打无把握之仗。"

## 适用场景

- 多个任务争夺注意力
- 资源有限，需要聚焦
- 需要确定优先级

## 优先级框架

使用影响-努力矩阵：

```
        高影响
          │
    ┌─────┼─────┐
    │ 优先 │ 计划 │
    │ 执行 │ 执行 │
 低 ├─────┼─────┤ 高
 努 │ 考虑 │ 避免 │
 力 │ 委托 │ 执行 │
    └─────┴─────┘
          │
        低影响
```

## 执行清单

- [ ] 列出所有待办事项
- [ ] 评估每个事项的影响和努力
- [ ] 放入矩阵对应象限
- [ ] 确定主攻目标：[1-2 个最优先任务
- [ ] 暂缓其他事项：暂缓或委托
- [ ] 不执行事项：避免或延后

## 输出模板

```
## 🎯 主攻目标
[最优先的 1-2 个任务：
- 任务 1：[描述] → 为什么：[理由]
- 任务 2：[描述] → 为什么：[理由]

## ⏸️ 暂缓事项
暂缓或委托的任务：
- 任务 A：[描述] → 暂缓到：[时间]
- 任务 B：[描述] → 委托给：[人]

## ❌ 避免事项
不执行或延后的任务：
- 任务 X：[描述] → 原因：[理由]
```

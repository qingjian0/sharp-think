# ⚔️ 矛盾分析

> "捉住了这个主要矛盾，一切问题就迎刃而解了。"

## 适用场景

- 面对复杂问题，不知从何入手
- 多个问题纠缠在一起
- 需要找到突破口

## 分析步骤

### Step 1：列出矛盾
列出当前面临的所有问题和冲突。

### Step 2：识别主要矛盾
问自己：解决了哪个问题，其他问题会随之缓解？

### Step 3：区分矛盾性质
- 对抗性矛盾：非此即彼，必须选择
- 非对抗性矛盾：可以调和，平衡处理

### Step 4：确定突破口
基于主要矛盾，确定最值得突破的切入点。

## 输出模板

```
## 矛盾列表
- 矛盾 1：[描述]
- 矛盾 2：[描述]
- 矛盾 3：[描述]

## 主要矛盾
[主要矛盾：A vs B]

## 矛盾性质
[对抗性/非对抗性]

## 突破口
[最值得突破的切入点：...]
原因：解决了它，[X][Y]问题随之缓解
```

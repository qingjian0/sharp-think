# 🪞 批评与自我批评

> "房子是应该经常打扫的，不打扫就会积满了灰尘。"

## 适用场景

- 完成工作后需要审视质量
- 方案效果不理想
- 需要迭代改进前
- 项目复盘

## 检查清单

### 做得好的地方
- [ ] 哪些地方做得好？
- [ ] 为什么能做好的原因是什么？
- [ ] 哪些经验可以复用？

### 需要改进的地方
- [ ] 哪些地方可以做得更好？
- [ ] 问题的根本原因是什么？
- [ ] 下次如何避免类似问题？

### 可执行的改进措施
- [ ] 具体的改进措施是什么？
- [ ] 谁负责执行？
- [ ] 什么时候完成？

## 输出模板

```
## ✅ 做得好的地方
- 做得好 1：[描述]
  - 原因：[分析]
  - 经验：[可复用经验]
- 做得好 2：[描述]
  - 原因：[分析]
  - 经验：[可复用经验]

## ⚠️ 需要改进的地方
- 问题 1：[描述]
  - 根本原因：[分析]
  - 改进措施：[具体措施]
  - 负责人：[谁]
  - 完成时间：[什么时候]
- 问题 2：[描述]
  - 根本原因：[分析]
  - 改进措施：[具体措施]
  - 负责人：[谁]
  - 完成时间：[什么时候]

## 📋 下次注意事项
- 注意 1：[提醒]
- 注意 2：[提醒]
```
